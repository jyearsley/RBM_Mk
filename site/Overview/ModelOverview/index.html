Model overview

RBM is a one-dimensional stream-temperature model that solves the time-dependent equation for the transfer of thermal energy across the air-water interface (Yearsley, 2009, 2012). The model uses a semi-Lagrangian, particle tracking numerical scheme that is highly scalable in space and time. The model software was written initially for purposes of developing the Total Maximum Daily Load (TMDL) for water temperature in the Columbia River system (Yearsley, 2003). The model had been modified to make use of the output from the large-scale hydrologic model VIC and regional scale model DHSVM.
